## 项目简介

Bunny Bites · 兔兔小厨房是一款基于 Moonbit 语言构建的轻量级在线点餐系统，以可爱的兔子主题为用户提供沉浸式的餐饮点餐体验。项目后端核心逻辑使用 Moonbit 编写，前端采用现代化响应式设计，涵盖菜品管理、购物车、订单处理等完整点餐流程。系统面向小型餐饮商户，提供从菜单展示、用户点餐到订单管理的一站式解决方案。

## 核心功能范围

- **菜品管理模块**：基于 Moonbit 实现菜品的结构化定义与管理，支持菜品分类（主食/甜品/饮品/沙拉/限定）、增删改查及 CSV 配置导入
- **购物车引擎**：使用 Moonbit 实现购物车核心逻辑，包括添加/删除商品、数量调整、价格计算及状态持久化
- **订单处理系统**：基于 Moonbit 实现订单的创建、状态流转（`pending → preparing → completed / cancelled`）及历史记录管理
- **RESTful API 服务**：提供基于 HTTP 的 JSON API，支持前端与后端的数据交互
- **前端交互界面**：采用 Tailwind CSS + 自定义主题的兔子风格响应式 Web 界面，包含菜品浏览、分类筛选、购物车面板、结算下单等完整交互流程
- **数据持久化**：支持菜品数据与订单数据的本地存储及 CSV 文件导入导出

## 技术栈

- **后端语言：** Moonbit
- **前端技术：** HTML5 / Tailwind CSS / JavaScript（ES6+）
- **API 协议：** RESTful JSON API
- **数据格式：** CSV / JSON
- **构建工具：** Moonbit CLI 工具链

## 项目亮点

1. **全 Moonbit 后端**：核心业务逻辑全部使用 Moonbit 语言实现，展示 Moonbit 在 Web 应用开发中的实战能力
2. **高颜值 UI**：以兔子主题为核心的可爱风格界面，提供愉悦的用户体验
3. **完整业务流程**：覆盖从菜单浏览到订单完成的完整点餐闭环
4. **模块化设计**：清晰的模块划分，便于后续功能扩展与维护
5. **数据驱动**：通过 CSV 配置文件管理菜品数据，降低运营维护成本
